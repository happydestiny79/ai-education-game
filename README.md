# Raise Your AI

Raise one AI companion across a **six-day main story**, then optionally keep going in **Extended Mode**. Your coaching shapes **Helpfulness, Accuracy, Bias, and Confidence** — moods, consequence flags, Day 6 climax choices, and endings.

## Run

Open `index.html` in a browser (HTTPS recommended for PWA files).

## Main story (Days 1–6)

| Feature | Detail |
|--------|--------|
| **6 days** | Structured arc: 3 + 3 + 2 + 2 + 2 + **3 finale** scenarios |
| **Day 6 climax** | High-stakes scenes that echo flags and active moods; choices set `climaxChoice` and can override earlier endings |
| **Risk & consequences** | Risky/gamble choices, temporary moods, flags that patch later scenes |
| **Living companion** | Emoji, aura, growth ring, stages Hatchling → Transcendent |
| **Endings** | Climax endings (accountability, redemption, walk-away, fallen megaphone) plus stat/flag-based outcomes |

## Extended Mode (optional)

After Day 6 and the main ending screen, choose **Enter Extended Mode** — or skip and raise another AI.

- No day cap; random scenarios (reused pool with light variation)
- Long-term goals panel (stat peaks, calm streaks, balanced runs, narrative “listen” goal, cross-run anthology)
- Richer legacy retire screen with key memory line and stat visuals
- **Retire** anytime for a legacy summary
- Moods, flags, and stats stay active

Save key: `raise_your_ai`
